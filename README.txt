HOW TO RUN
-----------------
1. Save the Code:
   Save the Python script as puzzle_solver.py.

2. Install Dependencies:
   Open your terminal or command prompt and run the following command:
   pip install psutil

3. Execute the Script:
   Run the program from your terminal:
   python puzzle_solver.py

4. Provide Input:
   The program will display a ">" prompt. Enter the 16 numbers of your puzzle (using 0 for the blank space), separated by single spaces, and press Enter. The numbers should represent the board from left-to-right, top-to-bottom.


EXAMPLE USAGE
-----------------

Below is a sample interaction with the program.

Input Prompt and User Entry:
> 1 0 2 4 5 7 3 8 9 6 11 12 13 10 14 15

Program Output:
Moves: RDLDDRR
Number of Nodes expanded: 361
Time Taken: 0.015

(Note: Time and Memory usage may vary slightly depending on your system.)
